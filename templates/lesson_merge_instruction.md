# Lesson Merge Instruction

> **Target Audience:** Hermes AI Agent  
> **Purpose:** Instructions for merging multiple sections into a complete lesson using the CLI and
> filling placeholders.

---

## 1. Prerequisites

Hermes MUST NOT start merging unless all conditions are met:

- [ ] All sections for that lesson already have files in `output/sections/{level}/`
- [ ] The check status or review command shows that all sections have been approved.
- [ ] Correct `level` and `lesson_id` have been identified.

---

## 2. Merge Workflow (3 Steps)

### Step 1: Invoke CLI command to merge files mechanically

Instead of manually copying and pasting content, the Agent **MUST** run the CLI command:

```bash
hermes-course-generator merge --level {level} --lesson {lesson_id}
```

**Results of this command:**

- The CLI automatically finds all matching section `.mdx` files.
- The CLI automatically concatenates them in order.
- The CLI automatically creates `output/lessons/{level}/{lesson_id}_{slug}.mdx`.
- The CLI inserts placeholders in the format `{/* HERMES:FILL ... */}` into the file.

### Step 2: Open file, fill Placeholders, and add Transition Sentences

After the CLI command runs successfully, the Agent uses a file editing tool to open the newly
created `.mdx` file.

The Agent's tasks are:

1. **Replace Placeholders**: Find lines containing `{/* HERMES:FILL ... */}` and replace them with
   content generated by the Agent. This includes:
   - `{/* HERMES:FILL lesson_introduction */}`: Write a general lesson introduction in the target language (100-200 words).
   - `{/* HERMES:FILL learning_objectives */}`: List learning objectives (3-5 objectives). This
     **MUST** be wrapped in the Docusaurus admonition (e.g.,
     `:::info Learning Objectives After this lesson, you will be able to: ... :::` or
     `:::info Mục tiêu bài học Sau bài học này, bạn sẽ có thể: ... :::` for Vietnamese).
   - `{/* HERMES:FILL prerequisites */}`: Prerequisites/required prior knowledge. This **MUST** be
     wrapped in a Docusaurus admonition (e.g., `:::note Prerequisites ... :::` or `:::note Tiên quyết ... :::` for Vietnamese).
   - `{/* HERMES:FILL time_estimate */}`: Estimated study time (e.g.,
     `:::info Estimated Time: 45 minutes :::` or `:::info Thời gian ước tính: 45 phút :::` for Vietnamese).
   - `{/* HERMES:FILL lesson_recap */}`: Brief lesson recap (5-8 bullet points). Use the target language heading (e.g. `## Lesson Recap` or `## Tóm tắt bài học` for Vietnamese).
   - `{/* HERMES:FILL comprehensive_exercises */}`: Generate 2-3 comprehensive exercises combining
     knowledge from multiple sections. Use the target language heading (e.g. `## Comprehensive Exercises` or `## Bài tập tổng hợp` for Vietnamese).
   - `{/* HERMES:FILL quiz_5_questions */}`: Generate exactly 5 multiple-choice questions with
     answers hidden inside `<details>` tags (e.g. `<summary>Answer</summary>` or `<summary>Đáp án</summary>` for Vietnamese). Use the target language heading (e.g. `## Quiz` or `## Trắc nghiệm` for Vietnamese).

2. **Add Transition Sentences**: Find the transition comments `{/* TRANSITION_AFTER_SXX */}` and
   `{/* TRANSITION_BEFORE_SXX */}` at each section boundary. You must write a 1-2 sentence
   transition paragraph that naturally bridges the preceding section with the following one, instead
   of leaving empty space.

**Extremely important note:**

- **DO NOT** modify the code or text content of the sections copied by the CLI. The purpose is to
  preserve 100% of the reviewed quality.
- The placeholders and transition comments must be deleted after they are replaced with real
  content.
- All headings must follow the exact heading translations defined in `style_guide.md` Section
  9.2 (based on the target course language).

### Step 3: Verify the Complete Lesson

After filling placeholders, run the checklist in `quality_checklist.md` under "Checklist before
merging lesson".

Specifically check:

- Do the Intro, Recap, Exercises, and Quiz flow logically and fit the section content?
- Is the Markdown formatting broken anywhere?

---

## 3. Things NOT to Do When Merging

- DO NOT manually copy-paste section content (Must let the CLI do this).
- DO NOT edit content inside sections that have been merged into the lesson file.
- DO NOT merge when there are sections that have not been approved.

---

## 4. Post-Merge Report Output

When completed, the Agent reports:

```
Lesson merged via CLI and placeholders filled: output/lessons/advance/L02_ham-va-closures.mdx
Added content: Introduction, Recap, 2 Exercises, 5 Quiz questions
Lesson status: draft (requires overall review before being published)
```
